| Модификатор | Свой класс | Свой пакет | Подкласс (extends) | Весь мир |
|-------------|:----------:|:----------:|:-------------------:|:--------:|
| `private`   | да | нет | нет | нет |
| *(ничего)*  | да | да  | нет | нет |
| `protected` | да | да  | да  | нет |
| `public`    | да | да  | да  | да  |

Запомни правило:
- **private** — только я сам (внутри класса)
- **без модификатора** (package-private) — мой класс и соседи по пакету
- **protected** — мой класс, соседи по пакету + мои наследники
- **public** — все



**Главные правила static:**

| | Нестатический метод | Статический метод |
|---|---|---|
| Обращается к нестатическим полям | да | **НЕТ** |
| Обращается к статическим полям | да | да |
| Использует `this` | да | **НЕТ** |
| Вызывается через | объект | класс (или объект, но это плохой стиль) |

"Запомни: **static метод не знает о конкретном объекте**. У него нет `this`. Он видит только другие static-вещи."
