**Условие:**
Создайте приложение, которое:
1. Вводит зарплаты двух сотрудников с клавиатуры
2. Определяет, у кого зарплата выше
3. Выводит большую зарплату в еврофомате: разделитель тысяч, 2 знака, символ €



## Задача 2: Static-методы и передача по значению

**Условие:**
Создайте 4 статических метода: `add`, `subtract`, `multiply`, `divide`.

Каждый метод:
- Получает два числа (number1 и number2)
- Выполняет операцию
- Сохраняет результат в number1 внутри метода
- Возвращает результат

В main: создайте number1 = 10, number2 = 5. Вызовите все 4 метода. **Убедитесь**, что после вызовов number1 и number2 НЕ изменились.

## ДЗ 5: Вероятность — красный и чёрный шар

Создайте класс `Probability` с методом, который:
- Имитирует 15 вытягиваний шара из мешка (2 шара: красный и чёрный)
- Использует `Math.random()` для каждого вытягивания
- Подсчитывает, сколько раз выпал красный и сколько чёрный
- Выводит процент каждого цвета с точностью до 2 знаков (DecimalFormat)

**Ожидаемый вывод (примерный):**
```
Draw 1: Red
Draw 2: Red
Draw 3: Black
...
Draw 15: Red
Red: 60.00%, Black: 40.00%
```

## ДЗ 6: Погода

Создайте класс `Weather` с методом, который генерирует случайную погоду на завтра:
- Вероятность: солнечно — 65%, дождь — 25%, снег — 10%
- Подсказка: `Math.random()` возвращает число от 0 до 1. Если < 0.65 → солнечно, если < 0.90 → дождь, иначе → снег.
- Запустите прогноз 10 раз и выведите результат.
