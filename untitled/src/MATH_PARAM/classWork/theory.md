Ключевые методы:"

```java
// Базовые
Math.abs(-410)        // 410
Math.max(10, 25)      // 25
Math.min(10, 25)      // 10

// Степень и корень
Math.pow(2, 3)        // 8.0 (2³)
Math.sqrt(25)         // 5.0

// Округление
Math.round(3.7)       // 4 (ближайшее целое)
Math.ceil(3.2)        // 4.0 (вверх)
Math.floor(3.8)       // 3.0 (вниз)

// Случайное число [0.0, 1.0)
Math.random()         // 0.73482...
```
