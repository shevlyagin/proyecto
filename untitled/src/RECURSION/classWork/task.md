# ДЗ 1: Сумма элементов массива — рекурсивно (Упр. 18)

Создайте рекурсивный метод `sumArray(int[] arr, int n)`, который вычисляет сумму первых n элементов массива.

- Базовый случай: n == 0 → return 0
- Рекурсивный шаг: arr[n-1] + sumArray(arr, n-1)

В main: массив {3, 7, 1, 9, 2}, вызвать sumArray(arr, arr.length).
